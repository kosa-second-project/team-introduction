# Downloaded tileset sources

This folder stores full original downloads for the extra map tile assets.

- `raw/`: Downloaded PNG/ZIP files.
- `extracted/`: Expanded ZIP contents for browsing and selection.

The curated, ready-to-review subset lives in `assets/map/tiles/extra/`.

Some early direct-download candidates remain in `raw/` for comparison only. Prefer the curated files in `assets/map/tiles/extra/`, where source pages and licenses are documented.
