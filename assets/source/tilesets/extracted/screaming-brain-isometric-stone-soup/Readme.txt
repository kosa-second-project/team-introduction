Isometric Stone Soup by Screaming Brain Studios (CC0)

This pack contains several thousand tiles that have been converted from the Dungeon Crawl 
Stone Soup (CC0) tileset collection.

Original Source: https://opengameart.org/content/dungeon-crawl-32x32-tiles-supplemental

Wall Tiles
==========
There are *.tsx tile-sheets for Tiled Map Editor for every tileset, which lets you mix and 
match tiles as needed. All Block tile-sheets also contain flat "Top Face" tiles which can be 
used as overlays on top of walls. These are found right beneath each of the Full and Half 
block tiles in every tile-sheet. This allows for hiding the top of certain blocks that may 
have decorations or features you only wish to be displayed on the sides of the block.

Floor Tiles
===========
All tile-sheets have been organized and named using the original Dungeon Crawl Stone Soup
categories and themes, and has a corresponding *.tsx tile-set for Tiled Map editor.
